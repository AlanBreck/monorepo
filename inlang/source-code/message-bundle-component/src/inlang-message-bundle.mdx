## Events

The message bundle component emits the following events:

- `change-message-bundle`: Emitted when the message bundle is changed. The event detail contains the new message bundle.
- `fix-lint`: Emitted when the fix lint button is clicked. The event detail contains the associated lint report and the fix id.
- `machine-translate`: Emitted when the machine translate button is clicked. The event detail contains the accociated messageId and variantId.

## Usage

- Solidjs - Prop: `prop:messageBundle={obj}` - Event: `on:change-message-bundle`
- Lit - Attribute: `messageBundle=${JSON.stringify(obj)}` - Event: `@change-message-bundle`
- Lit - Props: `.messageBundle=${obj}` - Event: `@change-message-bundle`
- Html - Attribute: `messageBundle=${JSON.stringify(obj)}` - Event: EventListener -> `change-message-bundle`
- React: Create react component: `createComponent` see: [@lit/react](https://lit.dev/docs/frameworks/react/#createcomponent)

### CSS Custom properties

The message bundle component uses the following css properties:

#### Primitives

From each property the component derives different shades and maps it on the shoelace components.

- `--inlang-color-primary`
- `--inlang-color-success`
- `--inlang-color-warning`
- `--inlang-color-danger`
- `--inlang-color-neutral`

#### Inputs

Most styles are derrived from input css properties.

Examples:
- `--sl-input-background-color`
- `--sl-input-background-color-hover`
- `--sl-input-border-color`
- `--sl-input-border-color-hover`
- `--sl-input-color`
- `--sl-input-color-focus`
- `--sl-input-color-hover`
- `--sl-input-placeholder-color`
- `--sl-input-background-color-disabled`
- `--sl-input-help-text-color`

#### Focus ring
- `--sl-input-focus-ring-color`
- `--sl-focus-ring-width`

#### ToolTip
- `--sl-tooltip-background-color`
- `--sl-tooltip-color`

#### Panels
- `--sl-panel-background-color`
- `--sl-input-help-text-color`
- `--sl-panel-border-color`

You can find all css properties listed [here](https://shoelace.style/tokens/more).
